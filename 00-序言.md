# 序言：深度学习简史

在进入具体技术之前，先了解深度学习的发展脉络。

## 关键里程碑

### 经典时代（1943-2011）

| 年份 | 事件 |
|------|------|
| 1943 | McCulloch-Pitts 神经元模型 |
| 1958 | Rosenblatt 的感知器 |
| 1986 | Rumelhart、Hinton、Williams 提出反向传播算法 |
| 1998 | LeNet-5（LeCun 等人的手写数字识别 CNN） |
| 2006 | Hinton 的深度置信网络（DBN）论文，标志着"深度学习"的复兴 |

### 深度学习革命（2012-2017）

| 年份 | 事件 |
|------|------|
| 2012 | AlexNet 在 ImageNet 竞赛中碾压传统方法，引爆深度学习革命 |
| 2013 | Word2Vec（词向量表示） |
| 2014 | GAN（生成对抗网络）；GoogLeNet/Inception；Seq2Seq |
| 2015 | ResNet（残差连接，突破 100 层） |
| 2016 | AlphaGo 战胜李世石（强化学习里程碑） |
| 2017 | Transformer（Attention Is All You Need）；AlphaGo 战胜柯洁 |

### 大模型时代（2018-至今）

| 年份 | 事件 |
|------|------|
| 2018 | BERT（双向语言模型）；GPT-1 |
| 2019 | GPT-2；EfficientNet |
| 2020 | GPT-3（1750 亿参数，涌现能力）；Vision Transformer (ViT) |
| 2021 | DALL-E（文本生成图像）；CLIP（多模态对齐） |
| 2022 | ChatGPT（RLHF 对齐）；Stable Diffusion；Whisper |
| 2023 | GPT-4（多模态大模型）；LLaMA（开源大模型）；Claude 2 |
| 2024 | Sora（视频生成）；Claude 3；Gemini；Llama 3 |
| 2025 | DeepSeek-R1；多模态 Agent；端侧模型爆发 |

## 核心概念分类

深度学习的众多概念可以从两个维度理解：

### 1. 训练与收敛（How to learn）

解决神经网络**如何从混沌中学习到正确权重**的问题，可细分为：

**学习机制** — 如何更新权重：
- **反向传播** — 计算梯度，让网络能"学习"
- **优化器**（SGD、Adam等）— 决定如何更新权重以收敛到最优解
- **损失函数**（MSE、交叉熵等）— 定义"什么是好"，指引优化方向
- **正则化** — 防止过拟合，提高泛化能力

**学习范式** — 用什么数据/信号来训练：
- **监督学习** — 用标签数据直接教
- **自监督学习** — 从数据本身构造监督信号
- **强化学习** — 从环境奖励信号学习
- **迁移学习** — 复用已有知识解决新问题

**数据与工程** — 让训练可行且高效：
- **数据预处理**（归一化、数据增强）— 准备合适的学习材料
- **分布式训练** — 解决大规模数据的训练效率问题
- **混合精度/梯度累积** — 显存与计算优化

### 2. 网络架构设计（What to build）

决定神经网络**应该设计成什么样**：

- **MLP** — 多层感知机，最基础的全连接结构
- **CNN** — 卷积神经网络，利用局部连接和权值共享处理图像
- **RNN** — 循环神经网络，处理序列数据，具备记忆能力
- **Seq2Seq** — 编码器-解码器架构，用于序列到序列的映射
- **残差连接** — 解决深层网络的梯度消失，让网络可以更深
- **注意力机制** — 让模型关注输入的重要部分

简单说：前者是"怎么学"，后者是"长什么样"。

### 训练与架构的互动

有趣的是，虽然一般人会认为**网络架构设计**更重要，但深度学习历史上屡次被**如何训练网络**的工程化问题所困扰：

- **反向传播**（1986）之前，神经网络受限于简单的单层感知器，无法训练深层网络
- **梯度消失**问题曾让研究者对深层网络望而却步，直到 **ReLU** 和 **残差连接** 等设计出现
- **大模型时代**，显存和计算瓶颈催生了 **混合精度训练**、**梯度检查点**、**分布式训练** 等工程创新
- **Transformer** 的成功，很大程度上得益于 **Adam 优化器** 和 **学习率调度** 的配合

可以说，**训练方法的突破往往反向推动架构设计的革新**——只有当你能够有效地训练一个网络时，那个架构才真正变得可行。两者相辅相成，缺一不可。
