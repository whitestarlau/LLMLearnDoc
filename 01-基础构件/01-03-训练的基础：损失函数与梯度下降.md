## 2. 损失函数：衡量预测好坏的标准

### 2.1 什么是损失函数？

**损失函数**衡量模型预测值与真实值之间的差距。

- 预测越准确，损失越小
- 预测越差，损失越大

训练神经网络的目标就是**最小化损失函数**。

### 2.2 常见的损失函数

**1. 均方误差（MSE）**——用于回归问题：

$$
L = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2
$$

其中 $y_i$ 是真实值，$\hat{y}_i$ 是预测值。

**2. 交叉熵（Cross-Entropy）**——用于分类问题：

$$
L = -\sum_{i=1}^{n} y_i \log(\hat{y}_i)
$$

### 2.3 直觉理解

```
预测值 vs 真实值：

真实值：猫（0）  狗（1）
预测1：猫（0.1）  狗（0.9）  → 损失小（预测准确）
预测2：猫（0.8）  狗（0.2）  → 损失大（预测错误）
```

### 2.4 为什么需要损失函数？

没有损失函数，我们就无法知道：

- 模型现在预测得怎么样
- 应该怎么调整参数来改进预测

**损失函数为训练提供了方向。**

### 2.5 损失函数的设计原理

#### 2.5.1 均方误差（MSE）为什么这么设计？

**1. 为什么用平方？**

| 特性                 | 作用                                                                      |
| -------------------- | ------------------------------------------------------------------------- |
| **惩罚大误差** | 平方会使大的误差变得更大（如误差2→4，误差4→16），迫使模型更关注严重错误 |
| **保证非负**   | 平方总是≥0，避免正负误差相互抵消                                         |
| **处处可微**   | 平方函数光滑，便于计算梯度                                                |

**2. 为什么用平均？**

- 使损失值与样本数量无关
- 便于比较不同规模数据集上的表现

**3. 数学本质**

最小化MSE等价于**最大化似然函数**，假设误差服从高斯分布：

$$
P(y|x,w) = \mathcal{N}(\hat{y}, \sigma^2)
$$

取对数似然后，最大化 $\sum \log P(y|x,w)$ 等价于最小化 $\sum (y - \hat{y})^2$

#### 2.5.2 交叉熵为什么这么设计？

**1. 信息论基础**

交叉熵衡量两个概率分布的差异：

- 当预测分布 $\hat{y}$ 与真实分布 $y$ 越接近，交叉熵越小
- 当两者完全相同时，交叉熵达到最小值

**2. 对数函数的特性**

```
log(x) 的图像：
    │
    │  ╲
    │   ╲
    │    ╲
    │     ╲___________
    └─────────────────→ x
    0    0.5    1
```

**关键特性**：

- 当 $\hat{y}_i \to 0$ 时，$-\log(\hat{y}_i) \to +\infty$
- 对错误预测施加**指数级惩罚**

**3. 直觉示例**

```
真实标签：猫（1）  狗（0）

预测1：猫（0.9）  狗（0.1）
损失 = -[1·log(0.9) + 0·log(0.1)] = -log(0.9) ≈ 0.105

预测2：猫（0.1）  狗（0.9）
损失 = -[1·log(0.1) + 0·log(0.9)] = -log(0.1) ≈ 2.303
```

预测2的损失是预测1的**22倍**！

**4. 为什么分类不用MSE？**

| 问题     | MSE                         | 交叉熵           |
| -------- | --------------------------- | ---------------- |
| 梯度消失 | 当预测接近0或1时，梯度趋近0 | 梯度始终合理     |
| 训练效率 | 慢，容易陷入局部最优        | 快，收敛稳定     |
| 概率解释 | 无                          | 自然对应概率分布 |

#### 2.5.3 设计哲学总结

| 损失函数         | 核心思想                         | 适用场景           |
| ---------------- | -------------------------------- | ------------------ |
| **MSE**    | 衡量"距离"，假设误差服从高斯分布 | 回归（预测连续值） |
| **交叉熵** | 衡量"分布差异"，基于信息论       | 分类（预测概率）   |

**共同原则**：

1. **可微性**：便于梯度下降
2. **非负性**：损失越小越好
3. **惩罚机制**：对严重错误施加更大惩罚
4. **统计基础**：与最大似然估计一致

## 3. 梯度：函数变化最快的方向

### 3.1 什么是梯度？

**梯度（Gradient）** 是一个向量，它指向函数增长最快的方向。

想象你站在山坡上，想知道往哪个方向走上升最快：

- 梯度就是指向山顶的方向
- 梯度的大小表示坡度的陡峭程度

在数学上，对于函数 $L(w)$，梯度就是导数 $\frac{\partial L}{\partial w}$。

### 3.2 直觉理解

```
损失函数 L(w) 的图像：

    L
    │    ╲
    │     ╲
    │      ╲
    │       ╲
    │        ● 当前位置
    │         ╲
    │          ╲
    └──────────────→ w

当前位置的梯度 ∂L/∂w < 0（斜率为负）
梯度指向函数增长最快的方向（向右）
```

---

## 4. 梯度下降：找到最小值的方法

### 4.1 梯度下降解决什么问题？

**核心问题**：我们有一个损失函数 $L(w)$，它衡量模型预测的好坏。我们希望找到一组参数 $w$，使得损失函数最小（即模型预测最准确）。

**挑战**：损失函数通常很复杂，有成千上万个参数，我们无法直接求解最小值在哪里。

**解决方案**：梯度下降是一种迭代优化算法，通过不断调整参数，逐步逼近最小值。

### 4.2 核心思想


假设我们有：

$$
L = (y - \hat{y})^2, \quad \hat{y} = wx + b
$$

其中：

- **$L$** 是损失函数（Loss），衡量预测值与真实值的差距
- **$y$** 是真实标签/真实值（ground truth）
- **$\hat{y}$** 是预测值（prediction），即模型的输出

梯度告诉我们：参数 $w$ **往哪个方向调、调整多少**，能让损失 $L$ 下降最多。我们求损失 $L$ 对参数 $w$ 的偏导数：

- 若 $\frac{\partial L}{\partial w} > 0$，说明增大 $w$ 会增大 $L$，所以要减小 $w$
- 若 $\frac{\partial L}{\partial w} < 0$，说明增大 $w$ 会减小 $L$，所以要增大 $w$

那么很容易知道：

- 梯度指向函数增长最快的方向
- 那么**梯度的反方向**就是函数下降最快的方向
- 沿着梯度的反方向更新参数，就能让损失函数越来越小

那么我们可以得到参数（权重）的更新公式：

$$
w \leftarrow w - \eta \cdot \frac{\partial L}{\partial w}
$$

其中：

- $w$：参数（权重）
- $\eta$：学习率（步长）
- $\frac{\partial L}{\partial w}$：损失函数对参数的梯度

### 4.3 直觉理解

```
损失函数 L(w) 的图像：

    L
    │    ╲
    │     ╲
    │      ╲
    │       ╲
    │        ● 当前位置
    │         ╲
    │          ╲
    └──────────────→ w

梯度 ∂L/∂w < 0（斜率为负）
所以 w ← w - η · (负数) = w + 正数
参数 w 会增大，沿着斜坡向下走
```

### 4.4 学习率的作用

- **学习率太小**：收敛太慢，需要很多步才能到达最小值
- **学习率太大**：可能跳过最小值，甚至发散
- **合适的学习率**：稳定快速地收敛到最小值

---

## 5. 从单个参数到多个参数

### 5.1 多参数的情况

对于多个参数 $w_1, w_2, \ldots, w_n$，梯度下降的更新公式变为：

$$
w_i \leftarrow w_i - \eta \cdot \frac{\partial L}{\partial w_i}, \quad i = 1, 2, \ldots, n
$$

每个参数都沿着自己梯度的反方向更新。

### 5.2 向量化表示

用向量表示：

$$
\mathbf{w} \leftarrow \mathbf{w} - \eta \cdot \nabla L(\mathbf{w})
$$

其中 $\nabla L(\mathbf{w})$ 是损失函数对所有参数的梯度向量。
