# 训练的基础：损失函数与梯度下降

## 1. 为什么需要这一章？

上一章我们学习了 MLP 的结构：多层神经元全连接。但还剩一个关键问题——**怎么训练它**？

在学习反向传播之前，需要先理解两个核心概念：
1. **损失函数**：衡量预测好坏的标准
2. **梯度下降**：如何调整参数来改进预测

---

## 2. 损失函数：衡量预测好坏的标准

### 2.1 什么是损失函数？

**损失函数**衡量模型预测值与真实值之间的差距。

- 预测越准确，损失越小
- 预测越差，损失越大

训练神经网络的目标就是**最小化损失函数**。

### 2.2 常见的损失函数

**1. 均方误差（MSE）**——用于回归问题：

$$
L = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2
$$

其中 $y_i$ 是真实值，$\hat{y}_i$ 是预测值。

**2. 交叉熵（Cross-Entropy）**——用于分类问题：

$$
L = -\sum_{i=1}^{n} y_i \log(\hat{y}_i)
$$

### 2.3 直觉理解

```
预测值 vs 真实值：

真实值：猫（0）  狗（1）
预测1：猫（0.1）  狗（0.9）  → 损失小（预测准确）
预测2：猫（0.8）  狗（0.2）  → 损失大（预测错误）
```

### 2.4 为什么需要损失函数？

没有损失函数，我们就无法知道：
- 模型现在预测得怎么样
- 应该怎么调整参数来改进预测

**损失函数为训练提供了方向。**

---

## 3. 梯度：函数变化最快的方向

### 3.1 什么是梯度？

**梯度（Gradient）** 是一个向量，它指向函数增长最快的方向。

想象你站在山坡上，想知道往哪个方向走上升最快：
- 梯度就是指向山顶的方向
- 梯度的大小表示坡度的陡峭程度

在数学上，对于函数 $L(w)$，梯度就是导数 $\frac{\partial L}{\partial w}$。

### 3.2 直觉理解

```
损失函数 L(w) 的图像：

    L
    │    ╲
    │     ╲
    │      ╲
    │       ╲
    │        ● 当前位置
    │         ╲
    │          ╲
    └──────────────→ w

当前位置的梯度 ∂L/∂w < 0（斜率为负）
梯度指向函数增长最快的方向（向右）
```

---

## 4. 梯度下降：找到最小值的方法

### 4.1 梯度下降解决什么问题？

**核心问题**：我们有一个损失函数 $L(w)$，它衡量模型预测的好坏。我们希望找到一组参数 $w$，使得损失函数最小（即模型预测最准确）。

**挑战**：损失函数通常很复杂，有成千上万个参数，我们无法直接求解最小值在哪里。

**解决方案**：梯度下降是一种迭代优化算法，通过不断调整参数，逐步逼近最小值。

### 4.2 核心思想

- 梯度指向函数增长最快的方向
- 那么**梯度的反方向**就是函数下降最快的方向
- 沿着梯度的反方向更新参数，就能让损失函数越来越小

更新公式：

$$
w \leftarrow w - \eta \cdot \frac{\partial L}{\partial w}
$$

其中：
- $w$：参数（权重）
- $\eta$：学习率（步长）
- $\frac{\partial L}{\partial w}$：损失函数对参数的梯度

### 4.3 直觉理解

```
损失函数 L(w) 的图像：

    L
    │    ╲
    │     ╲
    │      ╲
    │       ╲
    │        ● 当前位置
    │         ╲
    │          ╲
    └──────────────→ w

梯度 ∂L/∂w < 0（斜率为负）
所以 w ← w - η · (负数) = w + 正数
参数 w 会增大，沿着斜坡向下走
```

### 4.4 学习率的作用

- **学习率太小**：收敛太慢，需要很多步才能到达最小值
- **学习率太大**：可能跳过最小值，甚至发散
- **合适的学习率**：稳定快速地收敛到最小值

---

## 5. 从单个参数到多个参数

### 5.1 多参数的情况

对于多个参数 $w_1, w_2, \ldots, w_n$，梯度下降的更新公式变为：

$$
w_i \leftarrow w_i - \eta \cdot \frac{\partial L}{\partial w_i}, \quad i = 1, 2, \ldots, n
$$

每个参数都沿着自己梯度的反方向更新。

### 5.2 向量化表示

用向量表示：

$$
\mathbf{w} \leftarrow \mathbf{w} - \eta \cdot \nabla L(\mathbf{w})
$$

其中 $\nabla L(\mathbf{w})$ 是损失函数对所有参数的梯度向量。

## 6. 小结

| 概念 | 要点 |
|------|------|
| 损失函数 | 衡量预测好坏的标准，训练目标是最小化它 |
| 梯度 | 指向函数增长最快的方向 |
| 梯度下降 | 沿着梯度的反方向更新参数，逐步逼近最小值 |
| 学习率 | 控制每步更新的大小 |
