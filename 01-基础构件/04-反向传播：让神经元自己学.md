# 反向传播：让神经元自己学

## 历史背景：被遗忘的算法

### 1960-1970年代的困境

1969年，Minsky和Papert证明了单层感知机的局限性。研究者们意识到：**多层网络可能更强大**，但面临一个核心问题：

> **如何训练多层网络？单层感知机的学习规则无法直接推广到多层网络。**

当时的计算机学者们需要一个系统化的方法来计算"每个参数应该怎么调整"。

### 反向传播的早期发现

反向传播算法的思想其实很早就出现了：

- **1970年**：芬兰科学家 **Seppo Linnainmaa** 在他的硕士论文中首次描述了反向传播算法，但这篇论文没有引起广泛关注。
- **1974年**：美国科学家 **Paul Werbos** 在他的博士论文中独立发现了反向传播算法，并将其应用于神经网络，但同样没有得到重视。

### 1986年的复兴

1986年，**David Rumelhart**、**Geoffrey Hinton** 和 **Ronald Williams** 发表了里程碑式的论文：

> *"Learning representations by back-propagating errors"*

他们重新发现并推广了反向传播算法，证明了它在训练多层网络中的强大能力。这篇论文直接引发了神经网络研究的**第二次热潮**。

### 为什么叫"反向传播"？

因为梯度是从**输出层向输入层反向传播**的：

```
前向传播：输入 → 隐藏层 → 输出 → 损失
反向传播：损失 → 输出层 → 隐藏层 → 输入层（梯度）
```

---

## 1. 为什么需要反向传播

上一章我们学习了训练的基础：损失函数和梯度下降。但还剩一个关键问题——**如何高效计算多层网络中所有参数的梯度**？

对于单层网络，计算梯度很简单。但对于多层网络，参数有成千上万个，如何高效计算所有梯度？

**反向传播（Backpropagation）** 就是这个方法。

---

## 2. 核心思想：链式法则

### 2.1 一个简单例子

回顾上一章梯度下降我们讲过的例子：

对于单个神经元，假设我们有：

$$
L = (y - \hat{y})^2, \quad \hat{y} = wx + b
$$

其中：

- **$L$** 是损失函数（Loss），衡量预测值与真实值的差距
- **$y$** 是真实标签/真实值（ground truth）
- **$\hat{y}$** 是预测值（prediction），即模型的输出

我们需要调整的是权重参数 $w$ 。参数 $w$**往哪个方向调、调整多少**，能让损失 $L$ 下降最多。我们可以通过求损失 $L$ 对参数 $w$ 的偏导数：

- 若 $\frac{\partial L}{\partial w} > 0$，说明增大 $w$ 会增大 $L$，所以要减小 $w$
- 若 $\frac{\partial L}{\partial w} < 0$，说明增大 $w$ 会减小 $L$，所以要增大 $w$

更新规则：$w \leftarrow w - \eta \cdot \frac{\partial L}{\partial w}$（梯度下降）

更新规则中，只有$\frac{\partial L}{\partial w}$不是确定的（权重有初始化值、步长也是固定值）。

而想求 $\frac{\partial L}{\partial w}$，通过微积分知识，我们知道可以这样转换：

$$
\frac{\partial L}{\partial w} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial w}
$$

这就是**链式法则**——把复杂梯度拆成简单步骤。

### 2.2 计算图

把计算过程画成图：

```
x ──→ [×w] ──→ [+] ──→ ŷ ──→ [L] ──→ L
       ↑        ↑
       w        b
```

**前向传播**：从左到右计算值
**反向传播**：从右到左计算梯度

## 3. 反向传播的步骤

### 3.1 完整流程

```
1. 前向传播：从输入计算到输出，得到预测值
2. 计算损失：比较预测值和真实值
3. 反向传播：从损失出发，逐层计算梯度
4. 更新参数：w ← w - η · ∂L/∂w
```

### 3.2 数学推导

对于两层网络：

**前向传播：**

$$
z_1 = W_1 x + b_1 \quad \text{(隐藏层线性变换)}
$$

$$
a_1 = \sigma(z_1) \quad \text{(隐藏层激活)}
$$

$$
z_2 = W_2 a_1 + b_2 \quad \text{(输出层线性变换)}
$$

$$
\hat{y} = \sigma(z_2) \quad \text{(输出层激活)}
$$

$$
L = (\hat{y} - y)^2 \quad \text{(损失)}
$$

**反向传播：**

输出层梯度：

$$
\frac{\partial L}{\partial \hat{y}} = 2(\hat{y} - y)
$$

$$
\frac{\partial L}{\partial z_2} = \frac{\partial L}{\partial \hat{y}} \cdot \sigma'(z_2)
$$

$$
\frac{\partial L}{\partial W_2} = \frac{\partial L}{\partial z_2} \cdot a_1^T
$$

$$
\frac{\partial L}{\partial b_2} = \frac{\partial L}{\partial z_2}
$$

隐藏层梯度（链式法则继续传播）：

$$
\frac{\partial L}{\partial a_1} = W_2^T \cdot \frac{\partial L}{\partial z_2}
$$

$$
\frac{\partial L}{\partial z_1} = \frac{\partial L}{\partial a_1} \cdot \sigma'(z_1)
$$

$$
\frac{\partial L}{\partial W_1} = \frac{\partial L}{\partial z_1} \cdot x^T
$$

$$
\frac{\partial L}{\partial b_1} = \frac{\partial L}{\partial z_1}
$$

更新参数：

$$
W_1 \leftarrow W_1 - \eta \cdot \frac{\partial L}{\partial W_1}, \quad b_1 \leftarrow b_1 - \eta \cdot \frac{\partial L}{\partial b_1}
$$

$$
W_2 \leftarrow W_2 - \eta \cdot \frac{\partial L}{\partial W_2}, \quad b_2 \leftarrow b_2 - \eta \cdot \frac{\partial L}{\partial b_2}
$$

## 4. 激活函数：让网络"活"起来

### 4.1 什么是激活函数？

在上面的两层网络推导中，我们看到了激活函数的关键作用：

隐藏层：

$$
z_1 = W_1 x + b_1
$$

$$
a_1 = \sigma(z_1)
$$

输出层：

$$
z_2 = W_2 a_1 + b_2
$$

$$
\hat{y} = \sigma(z_2)
$$

**激活函数**就是这个 $\sigma$，它作用在每一层线性变换的结果上，产生该层的输出。

**为什么叫"激活"？** 这个名字来自生物神经元：当输入信号足够强时，神经元会"激活"（发放脉冲）。激活函数模拟了这个过程——决定神经元是否应该"激活"。

### 4.2 为什么需要激活函数？

**如果没有激活函数**，无论多少层网络：

$$
y = W_3(W_2(W_1 x + b_1) + b_2) + b_3 = W_{combined} \cdot x + b_{combined}
$$

**结果仍然是线性的！** 本质上和单层网络一样。

激活函数引入**非线性**，让网络能够学习复杂模式。这就是为什么：

- 单层感知机只能学线性问题（AND、OR）
- 多层网络 + 激活函数可以学非线性问题（XOR）

### 4.3 常见的激活函数

在反向传播中，我们需要激活函数的**导数**。不同的激活函数有不同的导数：

**Sigmoid**（历史最悠久）：

$$
\sigma(z) = \frac{1}{1 + e^{-z}}, \quad \sigma'(z) = \sigma(z)(1 - \sigma(z))
$$

- 输出范围：(0, 1)
- 导数最大值：0.25（导致梯度消失）

**ReLU**（现代最常用）：

$$
\text{ReLU}(z) = \max(0, z), \quad \text{ReLU}'(z) = \begin{cases} 1 & z > 0 \\ 0 & z \leq 0 \end{cases}
$$

- 计算简单，只需判断正负
- 正区间梯度恒为1，缓解梯度消失

**Tanh**（Sigmoid的改进）：

$$
\tanh(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}}, \quad \tanh'(z) = 1 - \tanh^2(z)
$$

- 输出范围：(-1, 1)，零中心
- 比Sigmoid好，但仍有梯度消失问题

### 4.4 激活函数在反向传播中的作用

在反向传播中，激活函数的导数是关键：

$$
\frac{\partial L}{\partial z} = \frac{\partial L}{\partial a} \cdot \sigma'(z)
$$

**这意味着**：

- 如果 $\sigma'(z)$ 很小（比如Sigmoid在饱和区），梯度会很小，参数更新慢
- 如果 $\sigma'(z) = 0$（比如ReLU在负区间），梯度完全消失，参数不再更新
- 如果 $\sigma'(z) = 1$（比如ReLU在正区间），梯度直接传递，训练高效

**这就是为什么选择激活函数如此重要**——它直接影响梯度能否有效传播。

### 4.5 为下一章做铺垫

在下一章"激活函数与权重初始化"中，我们将深入探讨：

- 各种激活函数的详细对比
- 如何根据网络结构选择合适的激活函数
- 权重初始化如何与激活函数配合

**核心思想**：激活函数和权重初始化是"一对搭档"——激活函数决定非线性能力，权重初始化决定信号能否稳定传播。

## 5. 计算图与自动微分

### 4.1 动态计算图

现代框架（PyTorch）使用动态计算图：

```python
import torch

x = torch.tensor([2.0], requires_grad=True)
w = torch.tensor([3.0], requires_grad=True)
b = torch.tensor([1.0], requires_grad=True)

# 前向传播（自动构建计算图）
z = w * x + b
a = torch.sigmoid(z)
loss = (a - 0.5) ** 2

# 反向传播（自动计算梯度）
loss.backward()

print(f"∂L/∂w = {w.grad}")  # 链式法则自动计算
print(f"∂L/∂b = {b.grad}")
```

### 4.2 自动微分的魔力

你只需要定义**前向计算**，框架自动处理：

- 构建计算图
- 链式法则求导
- 梯度累积

## 6. MLP 的反向传播

### 5.1 回顾网络结构

以上一章的两层 MLP 为例：

```
输入层    隐藏层    输出层
 x₁  ───→  h₁  ───→  y₁
    ╲    ╱ ╲    ╱
     ╲  ╱   ╲  ╱
     ╱  ╲   ╱  ╲
    ╱    ╲ ╱    ╲
 x₂  ───→  h₂  ───→  y₂
```

### 5.2 逐层反向传播

```python
# 前向传播
h = W1 @ x + b1          # 隐藏层线性变换
h_act = relu(h)          # 隐藏层激活
y = W2 @ h_act + b2      # 输出层线性变换

# 反向传播（从后往前）
dy = y - y_true                  # 输出层梯度
dW2 = dy @ h_act.T               # W2 的梯度
db2 = dy                         # b2 的梯度

dh_act = W2.T @ dy               # 传到隐藏层
dh = dh_act * (h > 0)            # ReLU 的导数

dW1 = dh @ x.T                   # W1 的梯度
db1 = dh                         # b1 的梯度

# 更新参数
W1 -= lr * dW1
b1 -= lr * db1
W2 -= lr * dW2
b2 -= lr * db2
```

## 7. 完整代码示例

本节代码示例演示用两层神经网络解决 **XOR（异或）问题**：

- 输入：(0,0)、(0,1)、(1,0)、(1,1)
- 期望输出：0、1、1、0（异或逻辑）

XOR 是经典非线性问题，单层感知机无法解决，但两层网络+激活函数可以学习。

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-np.clip(x, -500, 500)))

def sigmoid_derivative(x):
    s = sigmoid(x)
    return s * (1 - s)

class TwoLayerNet:
    def __init__(self, n_input, n_hidden, n_output):
        # Xavier 初始化
        self.W1 = np.random.randn(n_hidden, n_input) * np.sqrt(2.0 / n_input)
        self.b1 = np.zeros((n_hidden, 1))
        self.W2 = np.random.randn(n_output, n_hidden) * np.sqrt(2.0 / n_hidden)
        self.b2 = np.zeros((n_output, 1))
  
    def forward(self, X):
        """前向传播"""
        self.z1 = self.W1 @ X + self.b1  # @ = np.dot() 矩阵乘法。它是一个通过运算符重载实现的语法糖。
        self.a1 = sigmoid(self.z1)
        self.z2 = self.W2 @ self.a1 + self.b2
        self.a2 = sigmoid(self.z2)
        return self.a2
  
    def backward(self, X, Y):
        """反向传播"""
        m = X.shape[1]
  
        # 输出层梯度
        dz2 = self.a2 - Y  # 假设用 MSE 损失
        dW2 = (1/m) * dz2 @ self.a1.T
        db2 = (1/m) * np.sum(dz2, axis=1, keepdims=True)
  
        # 隐藏层梯度
        da1 = self.W2.T @ dz2
        dz1 = da1 * sigmoid_derivative(self.z1)
        dW1 = (1/m) * dz1 @ X.T
        db1 = (1/m) * np.sum(dz1, axis=1, keepdims=True)
  
        return dW1, db1, dW2, db2
  
    def train(self, X, Y, epochs=1000, lr=0.1):
        for epoch in range(epochs):
            # 前向传播
            output = self.forward(X)
  
            # 计算损失
            loss = np.mean((output - Y) ** 2)
  
            # 反向传播
            dW1, db1, dW2, db2 = self.backward(X, Y)
  
            # 更新参数
            self.W1 -= lr * dW1
            self.b1 -= lr * db1
            self.W2 -= lr * dW2
            self.b2 -= lr * db2
  
            if epoch % 100 == 0:
                print(f"Epoch {epoch}, Loss: {loss:.4f}")

# 测试 XOR 问题（单层感知机学不了，两层可以）
X = np.array([[0, 0, 1, 1], [0, 1, 0, 1]])
Y = np.array([[0, 1, 1, 0]])

net = TwoLayerNet(2, 4, 1)
net.train(X, Y, epochs=5000, lr=1.0)

print("\n预测结果：")
print(net.forward(X))
```

## 8. 常见问题

**Q: 梯度消失是什么？**

当网络很深时，梯度在反向传播中逐层相乘。如果每层的梯度都小于 1，最终梯度会趋近于 0：

$$
\frac{\partial L}{\partial W_1} = \frac{\partial L}{\partial a_n} \cdot \sigma'_n \cdot W_n \cdot \ldots \cdot \sigma'_1
$$

如果每层 $\sigma' \cdot W < 1$，乘积会指数衰减。

**Q: 梯度爆炸是什么？**

反过来，如果每层梯度都大于 1，梯度会指数增长，导致参数更新过大、训练不稳定。

**解决方案**：

- 使用 ReLU 替代 Sigmoid（ReLU 导数要么是 0 要么是 1）
- 梯度裁剪（Gradient Clipping）
- 残差连接（ResNet 的核心思想）

## 9. 小结

| 步骤     | 做什么                                                        |
| -------- | ------------------------------------------------------------- |
| 前向传播 | 从输入到输出，计算预测值                                      |
| 计算损失 | 衡量预测与真实的差距                                          |
| 反向传播 | 用链式法则计算每个参数的梯度                                  |
| 参数更新 | $w \leftarrow w - \eta \cdot \frac{\partial L}{\partial w}$ |
