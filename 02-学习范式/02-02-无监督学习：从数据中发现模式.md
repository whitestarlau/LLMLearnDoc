# 02-02 无监督学习：从数据中发现模式

上一章我们学习了监督学习——它依赖大量标注数据，模型从"问题和答案"中学习。但现实中，标注数据是稀缺且昂贵的。很多时候，我们只有原始数据，没有标签。

想象你收集了数百万用户的购物行为数据，想知道"这些用户可以分成哪几类"。你不知道有多少类别，也不知道每个用户属于哪一类。这时候，监督学习无能为力，但**无监督学习（Unsupervised Learning）** 可以帮我们从数据中自动发现隐藏的结构和模式。

无监督学习不需要标签，它从数据本身的结构中学习。这种能力让它成为数据探索、特征学习、异常检测等任务的有力工具。更重要的是，无监督学习是自监督学习（如GPT预训练）的基础。

本章将介绍无监督学习的核心方法：聚类、降维和密度估计。

---

## 1. 无监督学习的本质

### 1.1 什么是无监督学习？

**无监督学习 = 无标签的学习**

```
监督学习数据：
输入 (X)      标签 (Y)
───────────────────────
样本1    →    类别A
样本2    →    类别B
样本3    →    类别A

无监督学习数据：
输入 (X)      （没有标签）
───────────────────────
样本1    
样本2    
样本3    
...

目标：从数据中发现隐藏的结构、模式或表示
```

**"无监督"的含义**：没有老师告诉模型"正确答案是什么"，模型必须自己从数据中找出规律。就像让一个孩子看一堆图片，没人告诉他每张图是什么，他可能会自己发现"这些图看起来很像"、"那些图不太一样"。

### 1.2 无监督学习能做什么？

| 任务类型 | 目标 | 典型应用 |
|---------|------|---------|
| **聚类** | 将相似的数据分组 | 客户分群、新闻归类 |
| **降维** | 压缩数据，保留关键信息 | 数据可视化、特征压缩 |
| **异常检测** | 找出不正常的数据点 | 欺诈检测、故障预警 |
| **密度估计** | 学习数据的分布 | 生成新样本、数据增强 |

### 1.3 与监督学习的对比

```
监督学习：
  问：这张图是猫还是狗？
  答：是猫。（有标签告诉你答案）

无监督学习：
  问：这堆图片有什么规律？
  答：我发现它们可以分成3组，每组内部的图很相似。（自己发现）
```

| 方面 | 监督学习 | 无监督学习 |
|------|---------|-----------|
| 数据 | 有标签 | 无标签 |
| 目标 | 预测标签 | 发现结构 |
| 评估 | 准确率等指标 | 较难评估 |
| 应用 | 分类、回归 | 聚类、降维 |

## 2. 聚类：自动分组

### 2.1 K-Means 聚类

**核心思想**：将数据分成 K 个组，使组内数据尽可能相似

```
算法步骤：
1. 随机选择 K 个中心点
2. 将每个数据点分配到最近的中心
3. 重新计算每个组的中心
4. 重复 2-3 直到收敛
```

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans

# 生成示例数据
np.random.seed(42)
X = np.concatenate([
    np.random.randn(100, 2) + [0, 0],
    np.random.randn(100, 2) + [5, 5],
    np.random.randn(100, 2) + [0, 5]
])

# K-Means 聚类
kmeans = KMeans(n_clusters=3, random_state=42)
labels = kmeans.fit_predict(X)

# 可视化
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis')
plt.scatter(kmeans.cluster_centers_[:, 0], 
            kmeans.cluster_centers_[:, 1], 
            c='red', marker='x', s=200, label='Centers')
plt.title('K-Means Clustering')
plt.legend()
plt.show()
```

**如何选择 K？**

```python
# 肘部法则：画出不同 K 的损失曲线
inertias = []
K_range = range(1, 10)

for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42)
    kmeans.fit(X)
    inertias.append(kmeans.inertia_)

plt.plot(K_range, inertias, 'bo-')
plt.xlabel('K')
plt.ylabel('Inertia')
plt.title('Elbow Method for Optimal K')
plt.show()
# 寻找"肘部"位置，那是最佳的 K
```

### 2.2 层次聚类

**核心思想**：构建聚类的层次树状结构

```
自底向上（凝聚）：
1. 每个点是一个簇
2. 合并最相似的两个簇
3. 重复直到只剩一个簇

自顶向下（分裂）：
1. 所有点是一个簇
2. 分裂最不相似的簇
3. 重复直到每个点是一个簇
```

```python
from sklearn.cluster import AgglomerativeClustering
from scipy.cluster.hierarchy import dendrogram, linkage

# 层次聚类
hc = AgglomerativeClustering(n_clusters=3)
labels = hc.fit_predict(X)

# 绘制树状图
Z = linkage(X, 'ward')
plt.figure(figsize=(10, 5))
dendrogram(Z)
plt.title('Hierarchical Clustering Dendrogram')
plt.show()
```

### 2.3 DBSCAN：基于密度的聚类

**核心思想**：将密集区域视为簇，可以识别任意形状的簇

```
概念：
- ε-邻域：以某点为中心，半径为 ε 的区域
- 核心点：邻域内至少有 min_samples 个点
- 边界点：在核心点的邻域内，但自己不是核心点
- 噪声点：既不是核心点也不是边界点

优势：
- 不需要预先指定簇的数量
- 可以发现任意形状的簇
- 能识别噪声点
```

```python
from sklearn.cluster import DBSCAN

# DBSCAN 聚类
dbscan = DBSCAN(eps=0.5, min_samples=5)
labels = dbscan.fit_predict(X)

# -1 表示噪声点
n_clusters = len(set(labels)) - (1 if -1 in labels else 0)
print(f"发现 {n_clusters} 个簇")
```

## 3. 降维：压缩与可视化

### 3.1 为什么需要降维？

```
问题：高维数据
- 维度灾难：维度越高，数据越稀疏
- 计算成本高
- 难以可视化（人只能理解2D/3D）
- 可能包含冗余特征

目标：
- 压缩数据，保留关键信息
- 便于可视化
- 减少噪声
- 加速后续任务
```

### 3.2 PCA（主成分分析）

**核心思想**：找到数据方差最大的方向，投影到这些方向上

```python
from sklearn.decomposition import PCA

# 假设有100维的数据
X_high_dim = np.random.randn(1000, 100)

# PCA 降维到 2 维
pca = PCA(n_components=2)
X_low_dim = pca.fit_transform(X_high_dim)

print(f"原始维度: {X_high_dim.shape}")
print(f"降维后: {X_low_dim.shape}")
print(f"保留的方差比例: {pca.explained_variance_ratio_.sum():.2%}")
```

**可视化高维数据**：

```python
# 用 PCA 将数据降到 2D 进行可视化
import seaborn as sns

# 假设有一些带标签的高维数据
X = np.random.randn(200, 50)
y = np.random.randint(0, 3, 200)

pca = PCA(n_components=2)
X_2d = pca.fit_transform(X)

plt.figure(figsize=(8, 6))
scatter = plt.scatter(X_2d[:, 0], X_2d[:, 1], c=y, cmap='viridis')
plt.xlabel('PC1')
plt.ylabel('PC2')
plt.title('PCA Visualization')
plt.colorbar(scatter)
plt.show()
```

### 3.3 t-SNE：非线性降维

**核心思想**：保持高维空间中相似的点在低维空间也相似

```python
from sklearn.manifold import TSNE

# t-SNE 特别适合可视化
tsne = TSNE(n_components=2, random_state=42)
X_tsne = tsne.fit_transform(X_high_dim)

plt.scatter(X_tsne[:, 0], X_tsne[:, 1])
plt.title('t-SNE Visualization')
plt.show()
```

**PCA vs t-SNE**：

| 特性 | PCA | t-SNE |
|------|-----|-------|
| 类型 | 线性 | 非线性 |
| 速度 | 快 | 慢 |
| 保留信息 | 全局结构 | 局部结构 |
| 用途 | 特征压缩 | 可视化 |
| 可解释性 | 高（主成分） | 低 |

### 3.4 自编码器：神经网络降维

```python
import torch
import torch.nn as nn

class Autoencoder(nn.Module):
    def __init__(self, input_dim=784, latent_dim=32):
        super().__init__()
        
        # 编码器：压缩
        self.encoder = nn.Sequential(
            nn.Linear(input_dim, 256),
            nn.ReLU(),
            nn.Linear(256, 64),
            nn.ReLU(),
            nn.Linear(64, latent_dim)
        )
        
        # 解码器：重建
        self.decoder = nn.Sequential(
            nn.Linear(latent_dim, 64),
            nn.ReLU(),
            nn.Linear(64, 256),
            nn.ReLU(),
            nn.Linear(256, input_dim),
            nn.Sigmoid()
        )
    
    def forward(self, x):
        z = self.encoder(x)  # 压缩到潜在空间
        x_recon = self.decoder(z)  # 重建
        return x_recon, z

# 训练自编码器
model = Autoencoder()
criterion = nn.MSELoss()
optimizer = torch.optim.Adam(model.parameters())

# 模拟训练（实际使用真实数据）
for epoch in range(100):
    x = torch.randn(32, 784)  # 输入数据
    x_recon, z = model(x)
    loss = criterion(x_recon, x)  # 重建损失
    
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
```

## 4. 异常检测

### 4.1 基于密度的方法

```python
from sklearn.ensemble import IsolationForest

# Isolation Forest：孤立森林
iso_forest = IsolationForest(contamination=0.1, random_state=42)
outliers = iso_forest.fit_predict(X)

# -1 表示异常点，1 表示正常点
print(f"发现 {sum(outliers == -1)} 个异常点")
```

### 4.2 基于重构的方法

```python
# 自编码器用于异常检测
# 思路：正常数据能很好地重建，异常数据不能

def detect_anomaly(model, x, threshold):
    """
    如果重建误差 > threshold，则认为是异常
    """
    model.eval()
    with torch.no_grad():
        x_recon, _ = model(x)
        error = torch.mean((x - x_recon) ** 2, dim=1)
        return error > threshold
```

## 5. 无监督学习与深度学习

### 5.1 无监督预训练

在深度学习早期，无监督预训练是训练深度网络的关键技术：

```
流程：
1. 用无监督方法（如自编码器）预训练每一层
2. 各层学习到有用的特征表示
3. 在预训练基础上进行监督微调

现在：
- 大规模监督学习 + 迁移学习更主流
- 但自监督学习（无监督的一种）在 NLP 中重新崛起
```

### 5.2 与自监督学习的关系

```
无监督学习：完全没有标签
自监督学习：从数据中自动生成标签

例子：
- 自编码器（无监督）：重构输入
- GPT预训练（自监督）：预测下一个token（标签从数据中生成）

自监督学习可以看作是无监督学习的一种特殊形式
```

## 6. 核心要点总结

```
无监督学习 = 无标签的学习

┌─────────────────────────────────────────────────────────────┐
│  主要任务                                                   │
│  ├── 聚类：自动分组（K-Means, DBSCAN, 层次聚类）            │
│  ├── 降维：压缩与可视化（PCA, t-SNE, 自编码器）             │
│  └── 异常检测：找出异常点（Isolation Forest）               │
├─────────────────────────────────────────────────────────────┤
│  K-Means 聚类                                               │
│  ├── 需要指定 K（簇的数量）                                 │
│  ├── 简单高效，适合球形簇                                   │
│  └── 用肘部法则选择最佳 K                                   │
├─────────────────────────────────────────────────────────────┤
│  降维方法对比                                               │
│  ├── PCA：线性、快速、保留全局结构                          │
│  ├── t-SNE：非线性、慢、保留局部结构、适合可视化            │
│  └── 自编码器：非线性、可学习复杂特征                       │
├─────────────────────────────────────────────────────────────┤
│  优势                                                       │
│  ✓ 不需要标注数据                                           │
│  ✓ 能发现隐藏的数据结构                                     │
│  ✓ 适合探索性数据分析                                       │
├─────────────────────────────────────────────────────────────┤
│  局限                                                       │
│  ✗ 结果难以评估（没有标签）                                 │
│  ✗ 可能发现"无意义"的模式                                  │
│  ✗ 需要人工解释和验证                                      │
└─────────────────────────────────────────────────────────────┘

下一步：如何自动生成标签？→ 自监督学习
```
