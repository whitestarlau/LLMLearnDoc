# 02-03 自监督学习：自动生成监督信号

在前两章，我们学习了监督学习（需要人工标注）和无监督学习（完全没有标签）。有没有一种方法，既能享受监督学习的训练效率，又不需要人工标注呢？

这就是**自监督学习（Self-Supervised Learning）** ——它从数据本身自动生成标签，让模型在"自己创造的问题"上学习。

当你训练 GPT 预测"下一个词是什么"时，你不需要任何人告诉你答案——因为"下一个词"就隐藏在文本本身中。BERT 的"完形填空"任务也是如此：把句子中的词遮住，让模型猜是什么，答案就在原句里。

自监督学习是大模型时代的核心技术。GPT、BERT、CLIP 等模型都是通过自监督学习在海量无标注数据上预训练，学会了丰富的语言和视觉知识。本章将深入讲解自监督学习的原理和方法。

---

## 1. 自监督学习的本质

### 1.1 什么是自监督学习？

**自监督学习 = 自动生成标签的学习**

```
监督学习：
 人工标注 → "这张图是猫" → 训练模型

无监督学习：
 原始数据 → 没有标签 → 聚类、降维

自监督学习：
 原始数据 → 自动构造任务 → 自动生成标签 → 训练模型
 
例如：
 文本："今天天气很好，适合出门"
 遮盖："今天天气[MASK]，适合出门"
 标签："很好"（自动从原文获取）
```

**"自监督"的含义**：监督信号不是来自人工标注，而是从数据本身自动生成。模型自己给自己出题、自己核对答案。

### 1.2 与监督学习、无监督学习的关系

```
         监督学习
            ↑
            │ 有高质量人工标签
            │
     自监督学习 ──────────────────→
            │                       无监督学习
            │ 自动生成标签          完全无标签
            ↓
```

| 方面 | 监督学习 | 自监督学习 | 无监督学习 |
|------|---------|-----------|-----------|
| 标签来源 | 人工标注 | 自动生成 | 无 |
| 数据规模 | 受限于标注成本 | 可以无限大 | 可以无限大 |
| 训练效率 | 高（目标明确） | 高（有监督信号） | 较低 |
| 典型应用 | 分类、回归 | 预训练 | 聚类、降维 |

### 1.3 自监督学习的两个阶段

```
阶段一：自监督预训练
┌─────────────────────────────────────────────────┐
│  大规模无标注数据                                │
│       ↓                                         │
│  自动构造任务 + 自动生成标签                     │
│       ↓                                         │
│  训练模型学习通用表示                            │
└─────────────────────────────────────────────────┘

阶段二：监督微调
┌─────────────────────────────────────────────────┐
│  少量有标注数据                                  │
│       ↓                                         │
│  在预训练模型基础上微调                          │
│       ↓                                         │
│  适应具体任务                                    │
└─────────────────────────────────────────────────┘
```

这就是 GPT、BERT 等大模型的标准训练流程！

## 2. NLP 中的自监督学习

### 2.1 GPT：预测下一个词

**任务设计**：给定前面的词，预测下一个词

```
文本："今天天气很好，适合出去玩"

训练样本自动生成：
输入: "今天"           → 预测: "天气"
输入: "今天天气"       → 预测: "很好"
输入: "今天天气很好"   → 预测: "，"
输入: "今天天气很好，" → 预测: "适合"
...

标签从哪里来？原文本身就是标签！
```

**代码实现**：

```python
import torch
import torch.nn as nn

class GPTPretraining(nn.Module):
    """GPT 自监督预训练"""
    
    def __init__(self, vocab_size, d_model, n_layers):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, d_model)
        self.transformer = nn.TransformerDecoder(
            nn.TransformerDecoderLayer(d_model, nhead=8),
            num_layers=n_layers
        )
        self.lm_head = nn.Linear(d_model, vocab_size)
    
    def forward(self, input_ids):
        # 自回归：只能看到当前词之前的词
        embeddings = self.embedding(input_ids)
        hidden = self.transformer(embeddings, embeddings)
        logits = self.lm_head(hidden)
        return logits
    
    def compute_loss(self, input_ids):
        """
        输入: [batch, seq_len]
        目标: 输入向左移一位（预测下一个词）
        """
        logits = self.forward(input_ids[:, :-1])
        targets = input_ids[:, 1:]
        
        loss = nn.CrossEntropyLoss()(
            logits.reshape(-1, logits.size(-1)),
            targets.reshape(-1)
        )
        return loss

# 训练示例
model = GPTPretraining(vocab_size=50000, d_model=768, n_layers=12)
text = torch.randint(0, 50000, (2, 128))  # 模拟文本数据
loss = model.compute_loss(text)
print(f"预训练损失: {loss.item():.4f}")
```

**为什么有效**？要预测下一个词，模型必须学会：
- 语法结构
- 语义理解
- 常识推理
- 世界知识

### 2.2 BERT：完形填空

**任务设计**：随机遮盖一些词，让模型猜是什么

```
原文："今天天气很好，适合出去玩"

随机遮盖 15% 的词：
输入: "今天天气[MASK]，适合[MASK]去玩"
目标: "很好", "出"

标签从哪里来？原文！
```

**代码实现**：

```python
class BERTPretraining(nn.Module):
    """BERT 自监督预训练"""
    
    def __init__(self, vocab_size, d_model):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, d_model)
        self.encoder = nn.TransformerEncoder(
            nn.TransformerEncoderLayer(d_model, nhead=8),
            num_layers=12
        )
        self.mlm_head = nn.Linear(d_model, vocab_size)
    
    def forward(self, input_ids):
        # 双向：可以看到整个句子
        embeddings = self.embedding(input_ids)
        hidden = self.encoder(embeddings)
        logits = self.mlm_head(hidden)
        return logits
    
    def compute_loss(self, input_ids, masked_positions, masked_labels):
        """
        input_ids: 被遮盖后的输入
        masked_positions: 被遮盖的位置
        masked_labels: 被遮盖词的真实值（从原文获取）
        """
        logits = self.forward(input_ids)
        
        # 只计算被遮盖位置的损失
        masked_logits = logits[masked_positions]
        loss = nn.CrossEntropyLoss()(masked_logits, masked_labels)
        return loss
```

**BERT 的两个自监督任务**：

```
任务1：Masked Language Model (MLM)
- 随机遮盖 15% 的词
- 80% 替换为 [MASK]
- 10% 替换为随机词
- 10% 保持不变

任务2：Next Sentence Prediction (NSP)  
- 判断两个句子是否连续
- 正例：连续的句子
- 负例：随机拼接的句子
```

### 2.3 对比：GPT vs BERT

| 特性 | GPT | BERT |
|------|-----|------|
| **任务** | 预测下一个词 | 完形填空 |
| **方向** | 单向（从左到右） | 双向 |
| **架构** | Decoder | Encoder |
| **适用任务** | 文本生成 | 文本理解 |
| **自监督信号** | 下一个词 | 被遮盖的词 |

## 3. 视觉中的自监督学习

### 3.1 对比学习：SimCLR

**核心思想**：让同一张图片的不同增强版本相似，不同图片不相似

```
原始图片
    ├── 增强1（裁剪+颜色变换）──→ view1
    └── 增强2（裁剪+颜色变换）──→ view2

目标：
- view1 和 view2 相似（正样本对）
- view1 和其他图片的增强版本不相似（负样本对）
```

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class SimCLR(nn.Module):
    def __init__(self, backbone, projection_dim=128):
        super().__init__()
        self.backbone = backbone
        self.projection = nn.Sequential(
            nn.Linear(backbone.output_dim, 512),
            nn.ReLU(),
            nn.Linear(512, projection_dim)
        )
    
    def forward(self, x1, x2):
        # x1, x2 是同一张图片的两个不同增强
        h1 = self.backbone(x1)
        h2 = self.backbone(x2)
        
        z1 = self.projection(h1)
        z2 = self.projection(h2)
        
        return z1, z2
    
    def contrastive_loss(self, z1, z2, temperature=0.5):
        """
        对比损失：让正样本对相似，负样本对不相似
        """
        batch_size = z1.size(0)
        
        # 归一化
        z1 = F.normalize(z1, dim=1)
        z2 = F.normalize(z2, dim=1)
        
        # 拼接
        z = torch.cat([z1, z2], dim=0)  # [2*batch, dim]
        
        # 计算相似度矩阵
        sim = torch.mm(z, z.t()) / temperature  # [2*batch, 2*batch]
        
        # 正样本对的索引
        labels = torch.cat([
            torch.arange(batch_size, 2*batch_size),
            torch.arange(batch_size)
        ])
        
        # 损失
        loss = F.cross_entropy(sim, labels)
        return loss
```

### 3.2 MAE：遮盖 + 重构

**灵感来自 BERT**：遮盖部分图像，让模型重构

```
原始图像 (224×224)
    ↓ 随机遮盖 75% 的 patches
遮盖后图像 (只剩 25%)
    ↓ 编码器处理可见 patches
    ↓ 解码器重构完整图像
重构图像

标签从哪里来？原始图像！
```

```python
class MAE(nn.Module):
    """Masked Autoencoder for Vision"""
    
    def __init__(self, encoder, decoder, mask_ratio=0.75):
        super().__init__()
        self.encoder = encoder
        self.decoder = decoder
        self.mask_ratio = mask_ratio
    
    def forward(self, images):
        """
        1. 将图像分成 patches
        2. 随机遮盖一部分
        3. 编码可见 patches
        4. 解码重构完整图像
        """
        batch_size, channels, height, width = images.shape
        
        # 分 patch (简化版)
        patch_size = 16
        num_patches = (height // patch_size) * (width // patch_size)
        patches = images.unfold(2, patch_size, patch_size).unfold(3, patch_size, patch_size)
        patches = patches.reshape(batch_size, num_patches, -1)
        
        # 随机遮盖
        num_masked = int(num_patches * self.mask_ratio)
        mask = torch.rand(batch_size, num_patches).argsort(dim=1) < num_masked
        
        visible_patches = patches[~mask].reshape(batch_size, -1, patches.size(-1))
        
        # 编码 + 解码
        encoded = self.encoder(visible_patches)
        decoded = self.decoder(encoded)
        
        # 计算重构损失（只计算被遮盖的部分）
        loss = F.mse_loss(decoded, patches)
        
        return loss
```

### 3.3 CLIP：图文对齐

**核心思想**：让匹配的图文对相似，不匹配的不相似

```
数据：(图像, 文本描述) 配对

训练目标：
- 正确的图文对 → 相似度高
- 错误的图文对 → 相似度低

这是自监督吗？
- 图像和文本来自同一数据源（网页）
- "配对关系"自动提供了监督信号
- 可以看作弱自监督学习
```

## 4. 自监督学习的价值

### 4.1 为什么自监督学习如此重要？

```
传统监督学习的瓶颈：
1. 标注成本高
   - ImageNet 标注：数百万美元
   - 专业领域（医疗、法律）：更加昂贵

2. 数据规模受限
   - 有标注数据 ≈ 有限
   - 无标注数据 ≈ 无限（整个互联网）

3. 领域适应困难
   - 每个新任务都需要重新标注
   - 知识无法迁移
```

**自监督学习打破了这个瓶颈**：

```
自监督学习的优势：
1. 数据规模无限
   - 所有文本、图像、音频都可以用
   
2. 无需人工标注
   - 标签自动生成，成本几乎为零
   
3. 学到通用知识
   - 预训练模型可以迁移到各种任务
   
4. 涌现能力
   - 大规模预训练后，模型展现出意想不到的能力
```

### 4.2 预训练 + 微调范式

```
阶段一：预训练（Pre-training）
┌─────────────────────────────────────────────────┐
│  大规模无标注数据（TB 级文本）                   │
│       ↓                                         │
│  自监督学习（预测下一个词）                      │
│       ↓                                         │
│  学到通用的语言知识                              │
│       ↓                                         │
│  预训练模型（如 GPT-3）                          │
└─────────────────────────────────────────────────┘

阶段二：微调（Fine-tuning）
┌─────────────────────────────────────────────────┐
│  少量有标注数据（几千条）                        │
│       ↓                                         │
│  在预训练模型基础上训练                          │
│       ↓                                         │
│  适应具体任务（如客服机器人）                    │
└─────────────────────────────────────────────────┘
```

这就是现代大模型的训练范式！
