# AI Learn Doc

使用大模型引导生成的深度学习与大模型学习文档。

## 内容

从神经元基础出发，沿深度学习发展脉络，逐步讲解到现代大模型技术：

- **基础构件** - 神经元、反向传播、MLP、优化器
- **计算机视觉** - CNN、AlexNet
- **序列建模** - RNN、LSTM、Seq2Seq
- **注意力革命** - 注意力机制、Transformer
- **语言模型** - GPT、BERT
- **进阶专题** - LoRA、RLHF、Diffusion、多模态（**TODO**）
- **扩展阅读** - 其他AI流派与前沿话题
